================================================================================
                    AI EMAIL AUTOMATION - INSTALLATION GUIDE
================================================================================

================================================================================
PREREQUISITES
================================================================================
- Python 3.10 or higher
- Node.js 18 or higher
- Git installed on your system

================================================================================
STEP 1: BACKEND SETUP
================================================================================

1. Navigate to the backend folder:
   cd backend

2. Create a virtual environment (recommended):
   python -m venv venv

3. Activate the virtual environment:
   - Windows: venv\Scripts\activate
   - Mac/Linux: source venv/bin/activate

4. Install Python dependencies:
   pip install fastapi uvicorn imap-tools python-dotenv openai pandas
   pip install python-multipart email-validator

5. Create a .env file in the backend folder with your credentials:
   SMTP_HOST=smtp.gmail.com
   SMTP_PORT=587
   SMTP_USER=your-email@gmail.com
   SMTP_PASSWORD=your-app-password
   SMTP_FROM=your-email@gmail.com
   SMTP_USE_TLS=true
   IMAP_HOST=imap.gmail.com

   For Google Mail, generate an App Password:
   - Go to Google Account → Security
   - Enable 2-Step Verification
   - Go to App Passwords → Create new
   - Use the 16-character password above

================================================================================
STEP 2: FRONTEND SETUP
================================================================================

1. Navigate to the frontend folder:
   cd frontend

2. Install Node.js dependencies:
   npm install

3. Start the frontend development server:
   npm run dev

================================================================================
STEP 3: RUN THE APPLICATION
================================================================================

1. Start Backend (Terminal 1):
   cd backend
   uvicorn app.main:app --reload --port 8000

   Backend will run at: http://localhost:8000
   API Docs at: http://localhost:8000/docs

2. Start Frontend (Terminal 2):
   cd frontend
   npm run dev

   Frontend will run at: http://localhost:5173

================================================================================
USAGE GUIDE
================================================================================

INDIVIDUAL USERS:
- Register with your Gmail and 16-character App Password
- Use AI Agent to query your emails
- Go to "Send Emails" to send bulk emails

ORGANIZATIONS:
- Register as Organization
- Add sender accounts in "Senders" page
- Set active sender before sending

================================================================================
TROUBLESHOOTING
================================================================================

CORS Error:
- Make sure backend is running
- Check .env has valid SMTP credentials

Login Fails:
- Verify App Password is correct (16 lowercase letters)
- Check Google Account has 2-Step Verification enabled

Emails Not Sending:
- Verify SMTP credentials in .env
- Check App Password is valid

PROJECT STRUCTURE
================================================================================

  /src

    api.ts          - API client functions
    types.ts        - TypeScript type definitions
